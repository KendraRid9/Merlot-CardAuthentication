General error messages

# Error Meaning
1 {"error":{"message":"Not found"}} An invalid route is being used, cannot be found. The link that is being used is not using a valid route.
2 {"error":"GET Failed","message":"Expected different JSON data","paramsReceived":{}} A correct route is being used but no, or too few parameters are being used.
3 {"error":"Card ID not in database"} Correct route and parameters provided but incorrect values provided.
